# VK Token Checker

Android-приложение для проверки токенов VK и создания групповых звонков с современным Material Design 3 интерфейсом.

## 📱 Возможности

### Основные функции
- ✅ **Проверка токенов VK** - валидация токенов через VK API с отображением информации о пользователе
- 📞 **Создание групповых звонков** - генерация ссылок на звонки через выбранный токен
- 📊 **Статистика** - подсчёт валидных, невалидных токенов и ошибок
- 💾 **Автосохранение** - токены и результаты проверки сохраняются между запусками

### WebDAV Интеграция
- ☁️ **Загрузка токенов из WebDAV** - загрузка файла с токенами по прямому URL
- 🔐 **Basic Authentication** - поддержка логина и пароля для доступа к WebDAV
- 🔄 **Автосинхронизация** - автоматическая загрузка токенов при запуске приложения

### Интерфейс
- 🎨 **Material Design 3** - современный дизайн по стандартам Google
- 🌈 **3 цветовые темы**:
  - **Dynamic Color** - адаптация под обои устройства (Android 12+)
  - **Dark** - тёмная тема с сине-серой палитрой
  - **Light** - светлая тема
- 📱 **Edge-to-Edge** - полноэкранный режим
- 🔔 **Раскрывающиеся карточки** - подробная информация о каждом токене

## 🚀 Сборка и установка

### Требования
- Android Studio Hedgehog (2023.1.1) или новее
- JDK 17
- Android SDK 34

### Сборка через Android Studio
1. Откройте папку `VKTokenChecker` в Android Studio
2. Дождитесь синхронизации Gradle
3. Build → Build Bundle(s) / APK(s) → Build APK(s)

### Сборка через командную строку
```bash
cd VKTokenChecker
gradlew.bat assembleDebug
```

APK файл будет создан в:
```
app\build\outputs\apk\debug\app-debug.apk
```

## 📖 Использование

### Проверка токенов
1. Вставьте токены в текстовое поле (каждый с новой строки)
2. Нажмите кнопку **"Проверить"**
3. Дождитесь завершения проверки
4. Просмотрите результаты в списке

### WebDAV
1. Нажмите на иконку **📥** в верхней панели
2. Введите:
   - URL файла (например, `https://example.com/tokens.txt`)
   - Имя пользователя
   - Пароль
3. Нажмите **"Загрузить"**

**Быстрая загрузка**: Короткое нажатие на иконку 📥 загружает токены из сохранённых настроек WebDAV.

**Настройки WebDAV**: Долгое нажатие на иконку 📥 открывает диалог настроек.

### Автосинхронизация
1. Откройте настройки (иконка ⚙️)
2. Включите переключатель **"Автосинхронизация WebDAV"**
3. При следующем запуске токены загрузятся автоматически

### Создание звонка
1. После проверки токенов выберите нужный токен в списке **"Выберите токен для звонка"**
2. Нажмите **"Создать групповой звонок"**
3. Ссылка на звонок отобразится в карточке
4. Нажмите на иконку копирования для копирования ссылки

### Смена темы
1. Откройте настройки (иконка ⚙️)
2. Выберите цветовую тему:
   - **Dynamic Color** - адаптация под обои
   - **Dark** - тёмная тема
   - **Light** - светлая тема

## 🔐 Безопасность

- Токены хранятся только локально на устройстве
- WebDAV пароль сохраняется в зашифрованном SharedPreferences
- Приложение не требует специальных разрешений кроме доступа к интернету
- Cleartext трафик разрешён только для WebDAV

## 📝 Форматы данных

### Файл токенов
```
token1
token2
token3
```

### Сохранение результатов
Результаты проверки сохраняются в формате:
```
token1::Valid::12345::Имя Фамилия||token2::Invalid::Невалидный токен
```
